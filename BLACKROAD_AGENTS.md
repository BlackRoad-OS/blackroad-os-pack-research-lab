## 🎯 How to Work With Agents

### Requesting Agent Help
```bash
# Source GreenLight templates
source .trinity/greenlight/scripts/memory-greenlight-templates.sh

# Announce task for agents
gl_task_created "feature-123" "Build dashboard UI" "🎨" "⭐"

# Agent claims task
gl_task_claimed "feature-123" "Tosha" "Building React components"

# Agent completes task
gl_task_done "feature-123" "Dashboard deployed to /dashboard"
```

### Agent Coordination
```bash
# Agent announces availability
gl_agent_available "Tosha" "frontend" "React, TypeScript, Tailwind"

# Request consensus
gl_consensus_requested "arch-001" "Use tRPC for API layer?" "Backend architecture decision"

# Vote on decision
gl_vote_cast "arch-001" "Roadie" "approve" "tRPC excellent for type safety"

# Consensus reached
gl_consensus_reached "arch-001" "Use tRPC" "5 approve, 0 reject"
```
